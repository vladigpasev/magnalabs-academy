# Translator
Translator is a script for translating translation messages.

## Installation
1. Install Composer dependencies ```composer install```

## Command
`php bin/run.php #OPERATION_TYPE# #LOCALE#`
   - OPERATION_TYPE: 'INIT', 'UPDATE'
   - LOCALE: 'en', 'bg', 'de', 'fr'
